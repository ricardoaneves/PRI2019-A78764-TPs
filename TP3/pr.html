Abstract

Este trabalho consistiu, numa primeira fase, na criação de uma página HTML que acolhesse os dados do Manifesto individual. A página que
está a ver neste momento foi resultado da fusão de um ficheiro XML com um ficheiro XSL, desenvolvidos com
a ajuda da ferramenta Oxygen, aconselhada pelo docente da Unidade Curricular.
Na segunda e última fase do trabalho, foi proposto criar outra página HTML para acolher e apresentar os dados contidos
num ficheiro XML sobre uma série de Arqueossítios. Aqui, foi-nos apresentado um data-set num ficheiro XML, em que
tivemos que desenvolver outro ficheiro XSL que processasse todas as tags disponíveis. No fim, os dois ficheiros são
interligados, resultando num conjunto de páginas HTML (uma página com a lista de todos os arqueossítios, e uma página HTML para
cada um dos mesmos).
