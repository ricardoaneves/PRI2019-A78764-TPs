Abstract

Este trabalho consistiu na criação de um servidor para a gestão de alunos e as notas associadas aos mesmos.
Aqui, temos a possibilidade de visualizar os alunos já registados, registar (inserindo o seu número,
nome e curso)ou remover um aluno.
Para além disto, é possível também ver as notas de cada aluno. Para isto, basta clicar no botão "Ver notas",
onde o utilizador é redirecionado para a página do aluno. Aqui, é possíver adicionar, remover e visualizar as notas
do aluno em questão. Cada nota é constituída pelo seu identificador da Unidade Curricular e os valores obtidos.
