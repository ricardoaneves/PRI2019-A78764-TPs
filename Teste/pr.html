Abstract

Exercício 1
A persistência dos dados foi feita em MongoDB, sendo que foi necessário corrigir o ficheiro JSON
disponibilizador pelo docente da Unidade Curricular. Este ficheiro continha caractéres inválidos à importação
do ficheiro para a base de dados, pelo que tiveram de ser eliminados.
Corrigido o ficheiro JSON, foi importando com o seguinte comando:
mongoimport -d publications -c publications --jsonArray data/jcrpubs.json
Posto isto, a base de dados está populada e pronta a ser acedida através da API criada.
Para correr a resolução do exercício, terá que fazer download do servidor NodeJS e correr os seguintes comandos:
npm i
npm start
Para um melhor entendimento do resultado, aconselha-se o uso da ferramenta Postman.
