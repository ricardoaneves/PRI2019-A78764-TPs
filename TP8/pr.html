Abstract

Este trabalho consistiu na criação de um servidor para a guardar ficheiros, acompanhados de
uma pequena descrição. Aqui, a capacidade principal do sistema é a possibilidade de poder
fazer upload de vários ficheiros em simultaneo.
Para além disto, é possível ver e clicar em cada um dos ficheiros guardados. Se o
ficheiro se tratar de uma imagem (JPEG ou PNG), o sistema deverá apresentar a mesma,
através de uma janela que abre sob a tela.
