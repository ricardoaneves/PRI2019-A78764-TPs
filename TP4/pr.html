Abstract

Este trabalho consistiu na criação de um servidor para acolher cada uma
das músicas presentes no XML dado pelo docente da Unidade Curricular. Para isto,
foi necessário dividir o XML em vários XML individuais, e criar uma stylesheet geral.
Com isto, foi possível desenvolver o servidor, de modo a apresentar a página individual
da música passada como argumento no URL.
