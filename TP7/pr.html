Abstract

Este trabalho consistiu na criação de um servidor para a visualização de informações de várias obras.
Para isto, o docente da UC disponibilizou um ficheiro XML com toda esta informação, pelo que foi necessário
converter este ficheiro para JSON. De seguida, o dataset está pronto para ser importado para a base de
dados Mongo.
Assim, com este servidor, podemos ver a lista de todas as obras e os seus compositores, podendo, também, criar
uma série de queries úteis, tais como o ano de criação e o periodo da obra.
